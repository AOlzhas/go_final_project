# TODO Scheduler (Go Final Project)

## Описание проекта

Веб-приложение планировщика задач, написанное на языке Go.

Функции приложения:

- добавление задач
- редактирование задач
- удаление задач
- отметка задач выполненными
- поддержка повторяющихся задач

Данные хранятся в SQLite базе данных.
