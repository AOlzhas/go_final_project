# TODO Scheduler (Go Final Project)

## Описание проекта

Веб-приложение планировщика задач, написанное на языке Go.

Функции приложения:

- добавление задач
- редактирование задач
- удаление задач
- отметка задач выполненными
- поддержка повторяющихся задач

Данные хранятся в SQLite базе данных.

---
## Статус проекта

Проект завершён в рамках обязательной части задания.
Реализованы:
- веб-сервер
- SQLite база данных
- добавление задач
- список задач
- редактирование задач
- выполнение и удаление задач
- базовые правила повторения: `d N`, `y`
